# davidbuckley — Portfolio & Accountability System

**Deliverable of:** Phase 1, AI Career Pivot (see `C:/Users/David/AI Projects/GSD Sessions/AI Career/.planning/ROADMAP.md`)

## What this is
A Next.js 16 portfolio at `davidbuckley.vercel.app` that doubles as the accountability artifact for an 18-month self-taught pivot. Every page is a function of markdown files in `content/`. Progress bar counts checkboxes in `content/phases/*.md` at build time. Last-shipped timestamp + 30-day heatmap come from the GitHub API.

## Non-goals (v1)
- Custom domain (deferred to AAS-02)
- Tagging / search / RSS
- Dark mode toggle
- Analytics beyond Vercel defaults
- Blog/post infrastructure beyond the journal
