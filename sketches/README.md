# Sketches

Scratch code from study sessions. One file per session, runnable with Node.

```bash
npm run study:sketch          # creates today's sketch and opens in editor
node sketches/2026-04-16.mjs  # run a sketch
```

Multiple sketches on the same day get suffixed: `2026-04-16.mjs`, `2026-04-16-2.mjs`, `2026-04-16-3.mjs`.

These are learning artefacts — commit them. They are not part of the site build.
